WHAT I LEARNED:


1. Event ID 4625 in Windows Security Log tracks failed login attempts, with status code 0xC000006A specifically meaning "username correct but password wrong"


2. Source IP address is critical for threat assessment — 127.0.0.1 indicates local login attempts, while an external IP would suggest remote attack


3. Timestamps help distinguish between user error (rapid attempts within seconds as user retypes password) and automated brute force (consistent timing or thousands of attempts)


4. The MITRE ATT&CK framework maps failed login patterns to T1110 (Brute Force), but context (source IP, frequency, username patterns) determines actual threat level
